# WPGraphQL JWT Authentication

Dieses Verzeichnis enthält das WPGraphQL JWT Authentication Plugin (Third-Party).

## Hinweis

Kopiere das Plugin hierher:

```bash
cp -r /pfad/zu/wp-graphql-jwt-authentication-develop/* ./wp-graphql-jwt-authentication-develop/
```

## Quelle

Original: https://github.com/wp-graphql/wp-graphql-jwt-authentication

## Hinweis

Dieses Plugin benötigt keinen Build-Schritt und wird direkt in WordPress kopiert.
